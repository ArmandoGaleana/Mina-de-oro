# Mina-de-oro
código para recuperación de oro en una mina 

## En resumen, este proyecto se encuentra en el sector de la mineria. Y el código es aplicado para verificar que modelo de predicción es mejor para tener un error menor en la recuperación del oro en la mina.
## Se evaluaron dos modelos para compararlos, los cuales fueron LinearRegression y RandomForest, siendo este último el que mejores resultados obtuvo. Teniendo una predicción de error medio del 0.6%
